About AI Quiz Generator

Our Mission

In today's educational system, creating quality quizzes and exams takes significant time and resources. Our project aims to simplify this process by using artificial intelligence to automatically generate tests and quizzes based on educational materials.

The application allows professors to upload text documents, after which AI analyzes them and creates questions from different categories, including multiple-choice, true/false, and open-ended questions. The project is designed for higher education.

We believe that with this tool, the testing process will become faster, more efficient, and more objective, allowing professors to focus on improving teaching methods instead of spending time manually creating tests.

Key Features

  • User Roles: The system serves two types of users - professors and students.
  • Document Upload: Upload educational materials in PDF, DOCX formats or automatically load via Google Docs API.
  • Question Generation: Automatically generate multiple-choice, true/false, or essay questions.
  • Question Management: Review, edit, or delete questions before using them.
  • Admin Interface: A special interface for system administrators to view all users, uploaded documents, and generated tests.

How It Works

  1. Register an Account
    Create a professor account to access all features.
  2. Upload Educational Materials
    Upload your lecture notes, textbooks, or other educational content.
  3. Generate Questions
    Specify the number and types of questions you want to generate.
  4. Review and Edit
    Review the generated questions and make any necessary edits.
  5. Export and Use
    Export your quiz for use in your preferred learning platform.
